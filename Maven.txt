- Installation Process for MacOS
	- Install Java
		- Download jdk (Java SE Development Kit 8u202, i.e., jdk-8u202-macosx-x64.dmg) from Oracle website
		- Run the setup. Once completed, create environment variable
			- export JAVA_HOME="/Library/Java/JavaVirtualMachines/jdk1.8.0_301.jdk/contents/Home"
		- java -version
	- Download maven binary for Mac i.e. apache-maven-3.8.2-bin.tar.gz
	- Extract the tar in a folder.
	- Create Environment Variable.
		- cd ~
		- open .zshrc
		- add following lines
			- export JAVA_HOME="/Library/Java/JavaVirtualMachines/jdk1.8.0_301.jdk/contents/Home"
			- export M2_HOME=/volumes/Yasir/dev/apache-maven-3.8.2
			- export PATH=$PATH:$M2_HOME/bin
		- source .zshrc

- mvn clean
- mvn install -DskipTests
- mvn -Pdev
- mvn dependency:tree  //checks the dependencies added to your project
- mvn -Dspring.profiles.active=dev1,swagger,no-liquibase -Dspring.cloud.config.profile=dev1 -DskipTests=true
- mvn -version
- mvn --version
- mvn -v
